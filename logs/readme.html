Discord Message Logger Bot

Overview

This bot logs user activity and chat messages in a Discord server. It detects and flags concerning messages based on keyword detection and sentiment analysis.

Features

  • Logs usernames, user IDs, and account ages.
  • Tracks and records chat messages.
  • Detects concerning messages based on a predefined keyword list and sentiment analysis.
  • Logs flagged messages with context for review.

Usage

Simply add the bot to your Discord server, and it will start monitoring messages.

Logs Format

⚠️ Concerning Message Detected ⚠️
👤 User: [username]
📅 Time: [timestamp]
📌 Channel: [channel]
📍 Reason: [reason]
🔽 Context:
     💬 user: previous message
————————————————————————————————————————

Installation & Setup

  • Clone the repository.
  • Install dependencies: pip install -r requirements.txt
  • Run the bot and let it log messages in your server.
The bot only logs user messages and ignores bot messages. You can modify the concerning words list in database/concerningwords.txt.
